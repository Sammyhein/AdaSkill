# AdaSkill

Exercice de fin de bloc pour créer une page avec des compétences sur un thème choisie et un niveau d'expertise pour chaque compétences.

Dans un premier temps l'objectif était d'imaginer ce projet simplement avec du HTML/CSS tout en ajoutant des éléments qui nous sembleront utiles pour la suite.

Ensuite, la mission était de créer une base de donnée via le site Néon en pratiquant le langage SQL.

Après cette création, il était demandé de créer un dossier qui nous sert de BackEnd. Dans ce dossier, nous avons installer un bibliothèque Express.js. Nous avons connecté ce dernier avec Neon pour récuperer la base de donnée. Puis, nous avons créé nos routes afin de les manipuler sur le FrontEnd.

À la suite de cela, l'objectif était de remplacer le HTML initial et de le transformer en langage de code JavaScript tout en respectant le visuel de la version HTML.

Enfin, il a fallu rendre le site interactif en installant des boutons permettants d'ajouter des compétences et/ou des thèmes.
