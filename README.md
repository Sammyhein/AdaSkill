## 🇫🇷 Adaskill

**Adaskill** est un projet scolaire individuel full-stack ayant pour objectif de créer un site web présentant des compétences et leur niveau de progression, le tout relié à une base de données et modifiable depuis l'interface.

### 🎯 Objectifs du projet

- 📊 Affichage de **compétences** et de leur **niveau de progression**
- 🗄️ Données stockées et gérées depuis une **base de données**
- ✏️ Modification des compétences et niveaux directement depuis le **front-end**
- 🔧 Gestion des données via un **back-end** dédié

### 🛠️ Stack technique

- **Front-end :** HTML, CSS, JavaScript
- **Back-end :** Node.js, Express.js
- **Base de données :** SQL, hébergée sur [Neon](https://neon.tech/)

### 🌍 Langue

Le site est actuellement disponible **uniquement en français**.

### ⚠️ Version scolaire

> Ce projet a été réalisé individuellement dans un cadre scolaire. Il s'agit d'une version figée correspondant au rendu final du projet.

## 🇬🇧 Adaskill

**Adaskill** is an individual full-stack school project with the goal of building a website displaying skills and their progression levels, connected to a database and editable directly from the interface.

### 🎯 Project objectives

- 📊 Display of **skills** and their **progression levels**
- 🗄️ Data stored and managed in a **database**
- ✏️ Skills and levels editable directly from the **front-end**
- 🔧 Data management handled by a dedicated **back-end**

### 🛠️ Tech stack

- **Front-end:** HTML, CSS, JavaScript
- **Back-end:** Node.js, Express.js
- **Database:** SQL, hosted on [Neon](https://neon.tech/)

### 🌍 Language

The site is currently available **in French only**.

### ⚠️ School project

> This project was completed individually as part of a school assignment. It is a frozen version corresponding to the final project submission.
